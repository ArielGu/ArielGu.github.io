Module语法和加载方式

背景

Javascript 没有模块的(module)体系，ES6前主要是CommonJs(用于服务器端)、AMD(用于浏览器)

ES6 的加载方式：编译时加载/静态加载（即在编译时就完成模块加载），效率自然是比CommonJs（运行时加载）高。

ES6

1.import命令具有提升效果；

2.由于import是静态执行，所以不能使用变量和表达式（这种智能在运行时得到结果的语法结构）

3.import 语句会执行所加载的模块，但不会输入任何值，并且不会加载多次
